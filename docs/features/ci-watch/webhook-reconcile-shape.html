CI Watch Webhook Reconcile Shape

Webhooks are edge triggers. The backend does one authoritative reconcile for the watched PR head, transitions only on aggregate evidence, and treats GitHub's calculating mergeability state as pending instead of polling inside the webhook handler.

On narrow screens, lanes are stacked in execution order.
