Terminal Action

Reconcile only acts on aggregate state transitions. Red and conflict wake the agent; ready allows human merge or orchestration auto-merge; pending leaves the watch active.

Overview Agent handoff Webhook trigger Aggregate reconcile Terminal action
Red Set watch status=failed and wake the session to fix CI.
Conflict Set watch status=conflict and wake the session to rebase or resolve.
Ready Set watch status=ready for human merge or orchestration auto-merge.
Pending Refresh watch detail and last_event_at; keep status=watching.
